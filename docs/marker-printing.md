# Marker 10 Printing Instructions

1. Generate or open Marker ID `10` using the Original `ARUCO` dictionary.
2. Print the marker so the black square is exactly `30.0 cm` on each side.
3. Keep a white border around the full marker.
4. Turn off every print-scaling option, including "fit to page", "shrink oversized pages", and automatic margin adjustment.
5. Print on matte stock and mount it on a rigid flat board.
6. Measure the printed black square with a ruler after printing. Reprint if it is not exactly `30.0 cm`.

## Field notes

- A warped or glossy surface will reduce detection stability.
- If phone pose appears consistently too near or too far, verify the printed size again before tuning software.
- Use the same physical marker for all comparison runs so the error stays attributable to pose estimation rather than print variation.
